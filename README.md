# 🚀 AI Job Portal Frontend

A modern, responsive, and AI-powered Job Portal frontend built with **React**. This application helps users search jobs, optimize resumes, generate cover letters, improve LinkedIn profiles, prepare for interviews, and manage job applications through an intuitive user interface.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Muhammad-Khan-Khichi/AI-Driven-Recruitment-Frontend-
```

Go to project directory

```bash
cd ai-job-portal-frontend
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
VITE_API_URL=http://localhost:8000
```

Example:

```env
VITE_API_URL=https://your-api.com
```
